# Blood_donation
Developed a desktop application to manage donor and recipient data with CRUD operations and admin control, streamlining blood bank record management.

1. Admin Dashboard
- Displays the administrator’s control panel.
- Provides access to manage blood donation records, users, and system settings.
- Ensures smooth monitoring and management of the platform.
![image alt](https://github.com/GanyaPoojary/Blood_donation/blob/main/homepage_blood_donate.png)

2. CRUD Operations
- Demonstrates Create, Read, Update, and Delete functionalities.
- Admins can add new records, view existing ones, update details, or remove outdated entries.
- Core functionality for maintaining accurate blood donation data.
![image alt](https://github.com/GanyaPoojary/Blood_donation/blob/main/crud_operation.png)

3. User Add Blood (user_add_blood.png)
- Interface for users to register blood donations.
- Allows donors to input details such as name,blood group, address.
- Simplifies the process of contributing to the blood donation system
![image alt](https://github.com/GanyaPoojary/Blood_donation/blob/main/user_add_blood.png)
